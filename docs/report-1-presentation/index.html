Homepage Requirements

  1. Logged out user should be directed to a splash page.
  2. Logged in user should be presented with new posts

User Signup/Login Requirements

  1. Internet travelers should be able to create accounts on October easily by providing minimal information
    • username
    • email
    • password (Devise with bcrypt)
  2. Sample content upon signup in order to determine interest
  3. Able to link/reference a user's account in comments/posts similar to Twitter

User Profile Requirements

  1. All users have public profile pages
  2. Users will be able to add supplementary information not required for a profile here if they desire
  3. Will not show personally identifiable information (such as email) unless explicitly requested
  4. Will list submissions and comments from this user
